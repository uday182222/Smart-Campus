## 🚀 Two Simple Commands

### Command 1: Start Mobile App

**In your current terminal (you're in SmartCampusMobile):**

```bash

```

### Command 2: Start Backend (New Terminal)

**Open a NEW terminal window and run:**

```bash
cd /Users/udaytomar/Developer/Smart-Campus/server
npm run dev
```

## 📱 After Running npm start

You'll see:
```
› Metro waiting on exp://192.168.x.x:8081
› Scan the QR code above with Expo Go (Android) or the Camera app (iOS)

› Press a │ open Android
› Press i │ open iOS simulator
› Press w │ open web
```

**Then:**
- Press `i` → Opens iOS Simulator
- Press `a` → Opens Android Emulator  
- Scan QR → Opens on your phone (requires Expo Go app)
