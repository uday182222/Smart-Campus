## 🔧 BUILD COMMANDS

### **Create Production Build:**
```bash
cd /Users/udaytomar/Developer/Smart-Campus/SmartCampusMobile

# For APK (easier, works everywhere)
npx expo build:android -t apk

# For AAB (required by Play Store)
npx expo build:android -t app-bundle
```

### **Build with EAS (Recommended):**
```bash
# Install EAS CLI
npm install -g eas-cli

# Login to Expo
eas login

# Configure project
eas build:configure

# Build for Android
eas build --platform android
```

## 📋 SUBMISSION STEPS

### **STEP 1: Create App**
1. Go to Play Console
2. Click "Create app"
3. Enter app name: "Smart Campus"
4. Select language: English
5. Choose app or game: App
6. Choose free or paid: Free
7. Accept declarations
8. Click "Create app"

### **STEP 2: Set Up App**
1. Fill in all store listing details
2. Upload all required graphics
3. Select app category: Education
4. Add contact details
5. Save draft

### **STEP 3: Content Rating**
1. Go to "Content rating"
2. Enter email address
3. Select "Education"
4. Answer questionnaire honestly
5. Submit for rating
6. Wait for rating (instant)

### **STEP 4: Target Audience**
1. Select age groups
2. Add if app targets children
3. Confirm declarations

### **STEP 5: Data Safety**
1. Fill data collection form
2. Specify data types
3. Explain data usage
4. Describe security practices
5. Submit

### **STEP 6: App Access**
1. Provide demo credentials if needed
2. Explain special access requirements

### **STEP 7: Ads**
1. Declare if app contains ads (No)

### **STEP 8: Select Countries**
1. Choose "All countries" or select specific
2. Configure distribution

### **STEP 9: Upload App**
1. Go to "Production" release
2. Click "Create release"
3. Upload APK/AAB file
4. Add release notes
5. Save

### **STEP 10: Review & Publish**
1. Review all sections
2. Fix any issues highlighted
3. Submit for review
4. Wait for approval (1-7 days)
5. Once approved, app goes live!
