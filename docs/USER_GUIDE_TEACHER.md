## Getting Started

### First Time Login

1. **Download the App**
   - Download "Smart Campus" from App Store (iOS) or Google Play Store (Android)
   - Install and open the app

2. **Login Credentials**
   - You will receive your login credentials from your school administrator
   - Email: Your school-provided email address
   - Password: Temporary password (you'll be asked to change it on first login)

3. **Complete Your Profile**
   - Add profile picture
   - Verify contact information
   - Set notification preferences

### Navigation

- **Bottom Navigation Bar**: Quick access to main features
  - 🏠 **Dashboard**: Overview of your classes and tasks
  - 📊 **Attendance**: Mark and view attendance
  - 📚 **Homework**: Create and manage assignments
  - 📝 **Marks**: Enter and view student grades
  - ⚙️ **More**: Settings, profile, and additional features

## Mark Attendance

### How to Mark Attendance

#### Option 1: Quick Mark (Recommended)

1. **Select Class**
   - Tap on the class from dashboard
   - Or go to "Attendance" tab

2. **Choose Date**
   - Attendance defaults to today
   - Tap calendar icon to select a different date

3. **Mark Students**
   - All students listed with photos
   - Tap status buttons:
     - ✅ **Present** (Green)
     - ❌ **Absent** (Red)
     - ⏰ **Late** (Orange)
     - 🏠 **Half Day** (Yellow)

4. **Submit**
   - Tap "Submit Attendance" button
   - Confirmation message appears
   - Parents receive automatic notification

#### Option 2: Bulk Mark

1. **Select All Present**
   - Tap "Mark All Present" button
   - Manually change status for absent students

2. **Submit**
   - Review the summary
   - Tap "Submit" to save

### Edit Attendance

**To modify attendance after submission:**

1. Go to "Attendance" → "History"
2. Select the date
3. Find the student
4. Tap "Edit" icon
5. Change status and add notes
6. Save changes

### Attendance Reports

**View Reports:**
- **Daily Report**: Today's attendance summary
- **Weekly Report**: 7-day attendance overview
- **Monthly Report**: Full month statistics
- **Student-wise**: Individual student attendance history

**Export Options:**
- Download as PDF
- Share via email
- Export to Excel

### Tips & Best Practices

✅ **Do's:**
- Mark attendance by 10 AM daily
- Add notes for absences if known
- Review and confirm before submitting
- Check for duplicate entries

❌ **Don'ts:**
- Don't mark attendance for future dates
- Don't forget to submit after marking
- Don't mark attendance twice for the same day

## Manage Homework

### Create New Homework

1. **Navigate to Homework**
   - Tap "Homework" in bottom navigation
   - Or tap "Create Homework" from class dashboard

2. **Fill in Details**
   - **Class**: Select class (auto-selected if from class view)
   - **Subject**: Choose subject from dropdown
   - **Title**: Brief title (e.g., "Chapter 5 Exercises")
   - **Description**: Detailed instructions
   - **Due Date**: Select deadline
   - **Attachments** (Optional): Add files, images, or links

3. **Set Options**
   - **Priority**: Normal, Important, Urgent
   - **Marks**: Total marks for assignment
   - **Submission Type**: Photo, PDF, Text, None
   - **Visibility**: All students or specific students

4. **Publish**
   - Tap "Create Homework" button
   - Students and parents receive instant notification

### Edit Homework

1. Go to "Homework" → "My Assignments"
2. Find the homework
3. Tap three-dot menu → "Edit"
4. Make changes
5. Tap "Update" to save

### View Submissions

1. Tap on homework assignment
2. See submission status:
   - ✅ Submitted (Green)
   - ⏳ Pending (Orange)
   - ❌ Overdue (Red)

3. Review student submissions:
   - View submitted files
   - Add comments
   - Give feedback
   - Award marks

### Delete Homework

1. Find the homework
2. Tap three-dot menu → "Delete"
3. Confirm deletion
4. Note: Deletion notifies all students

### Homework Templates

**Save time with templates:**

1. Create homework normally
2. Tap "Save as Template"
3. Give template a name
4. Reuse template next time:
   - Tap "Use Template"
   - Select from saved templates
   - Modify if needed

## Enter Marks

### Create Exam/Test

1. **Navigate to Marks**
   - Tap "Marks" in bottom navigation

2. **Create New Exam**
   - Tap "➕ Create Exam"
   - Fill in details:
     - **Name**: "Mid-term Exam", "Unit Test 1"
     - **Subject**: Mathematics, Science, etc.
     - **Total Marks**: 100
     - **Date**: Exam date
     - **Type**: Unit Test, Mid-term, Final, Quiz

3. **Publish Exam**
   - Students can now see the exam in their schedule

### Enter Marks for Students

#### Option 1: Individual Entry

1. Select the exam
2. Tap on student name
3. Enter marks obtained
4. Add remarks (optional)
5. Tap "Save"

#### Option 2: Bulk Entry (Recommended)

1. Select the exam
2. Tap "Bulk Entry"
3. Enter marks for all students in a grid
4. Marks automatically validated against total
5. Tap "Submit All"

### Grade Calculation

**Automatic grading:**
- A+ : 90-100%
- A  : 80-89%
- B+ : 70-79%
- B  : 60-69%
- C  : 50-59%
- D  : 40-49%
- F  : Below 40%

Grades are calculated automatically based on percentage.

### Edit Marks

1. Go to exam details
2. Find student
3. Tap "Edit" icon
4. Change marks
5. Add reason for edit
6. Save (edit is logged in audit trail)

### View Analytics

**Class Performance:**
- Average marks
- Highest/lowest scores
- Grade distribution
- Subject-wise analysis

**Student Progress:**
- Individual student performance
- Comparison with class average
- Improvement trends

### Export Reports

- Download mark sheets
- Generate progress reports
- Share with parents
- Export to Excel

## Troubleshooting

### Can't Login?

**Check:**
- ✅ Email is correct (use school-provided email)
- ✅ Password is correct (case-sensitive)
- ✅ Internet connection is active
- ✅ App is updated to latest version

**Still not working?**
- Tap "Forgot Password"
- Contact school admin
- Check school IT support

### Attendance Not Submitting?

**Possible reasons:**
- Internet connection lost
- Attendance already marked for this date
- API server is down

**Solution:**
- Check internet connection
- Try again in a few minutes
- Contact IT support if issue persists

### Homework Not Creating?

**Check:**
- All required fields are filled
- Due date is in the future
- File attachments are < 10MB
- You have permission for selected class

### Marks Not Saving?

**Common issues:**
- Marks exceed total marks
- Exam date is in the future
- Network error during submission

**Solution:**
- Validate marks are within range
- Check exam date
- Retry submission

### App Running Slow?

**Performance tips:**
- Close unused apps
- Clear app cache (Settings → Storage)
- Update to latest version
- Restart device

### Lost Internet Connection?

**Offline mode:**
- App shows offline indicator
- Some features work offline:
  - View cached data
  - Mark attendance (syncs when online)
  - View past records

**When back online:**
- App automatically syncs pending actions
- You'll see a sync indicator

## FAQs

### Q: Can I mark attendance for past dates?
**A:** Yes, you can select a past date from the calendar and mark attendance. However, there may be a limit (e.g., 7 days back) set by your school.

### Q: How do I add a new student to my class?
**A:** Contact your school administrator. Only admins can add students to classes.

### Q: Can I edit marks after parents have viewed them?
**A:** Yes, but edits are logged in the audit trail with reason and timestamp.

### Q: What happens if I delete homework?
**A:** Homework is permanently deleted and all students/parents are notified. Submissions are also deleted.

### Q: Can I see other teachers' data?
**A:** No, you can only see data for classes you teach, ensuring privacy.

### Q: How do I download attendance reports?
**A:** Go to Attendance → Reports → Select date range → Download PDF/Excel

### Q: Can I create homework for multiple classes at once?
**A:** Not yet. You need to create homework separately for each class.

### Q: What file types can I attach to homework?
**A:** PDF, DOC, DOCX, JPG, PNG, MP4 (max 10MB per file)
