## Getting Started

### Download & Install

1. **iOS Users**
   - Open App Store
   - Search "Smart Campus"
   - Tap "Get" → Install
   - Open app

2. **Android Users**
   - Open Google Play Store
   - Search "Smart Campus"
   - Tap "Install"
   - Open app

### First Time Setup

1. **Login**
   - Email: Provided by school (usually parent email)
   - Password: Temporary password from school
   - Tap "Login"

2. **Change Password**
   - You'll be prompted to change password
   - Enter new secure password
   - Confirm password
   - Save

3. **Setup Profile**
   - Add profile picture (optional)
   - Verify phone number
   - Set emergency contact
   - Save

4. **Enable Notifications**
   - Allow push notifications
   - Get instant updates about your child
   - Configure notification preferences

## Check Attendance

### Today's Attendance

**View today's status:**
1. Check dashboard "Attendance" card
2. Status shown:
   - ✅ **Present** - Child attended school
   - ❌ **Absent** - Child was absent
   - ⏰ **Late** - Child arrived late
   - 🏠 **Half Day** - Child attended half day

3. **Marked At**: Time when attendance was recorded
4. **Marked By**: Teacher's name

### Attendance History

**View past attendance:**

1. Tap "Attendance" card on dashboard
2. Or go to "More" → "Attendance"

3. **Monthly View**
   - Calendar showing attendance days
   - Green = Present, Red = Absent, Orange = Late

4. **Statistics**
   - Total school days this month
   - Present days count
   - Absent days count
   - Attendance percentage

5. **Detailed View**
   - Tap any date to see details
   - See time marked
   - See teacher who marked
   - View any notes

### Attendance Reports

**Download reports:**
1. Go to Attendance → "Reports"
2. Select date range
3. Choose format (PDF or Excel)
4. Tap "Download"
5. Share or save to device

### Set Attendance Alerts

**Get notified daily:**
1. Settings → Notifications
2. Enable "Attendance Alerts"
3. Set preferred time (e.g., 10 AM)
4. Receive notification if child is absent

## View Homework

### Pending Homework

**See all pending assignments:**

1. Tap "Homework" from dashboard
2. Or go to "More" → "Homework"

3. **Homework List**
   - Subject
   - Title
   - Due date
   - Priority indicator
   - Submission status

4. **Tap on homework** to view:
   - Full description
   - Attachments (if any)
   - Submission deadline
   - Total marks

### Help Your Child Submit

**Submission types:**

- **Photo Upload**: Take picture of completed work
- **PDF Upload**: Upload scanned document
- **Text Entry**: Type answers directly
- **No Submission**: Just complete and teacher checks in class

**To submit:**
1. Open homework
2. Tap "Submit"
3. Choose submission method
4. Upload file or take photo
5. Add comments (optional)
6. Tap "Submit Homework"
7. Confirmation shown

### Check Submission Status

- ✅ **Submitted**: Work submitted successfully
- 📝 **Graded**: Teacher has graded it
- ⏳ **Pending**: Not yet submitted
- ❌ **Overdue**: Missed deadline

### Homework Notifications

**You'll receive notifications for:**
- New homework assigned
- Submission deadline reminder (24 hours before)
- Homework graded
- Teacher comments added

## Troubleshooting

### Dashboard Not Loading?

1. Check internet connection
2. Pull down to refresh
3. Logout and login again
4. Clear app cache
5. Reinstall app if issue persists

### Wrong Child's Data Showing?

1. Check selected child in dropdown
2. Tap child selector at top
3. Choose correct child
4. Data will refresh automatically

### Notifications Not Received?

**Check:**
1. Settings → Notifications → Ensure enabled
2. Phone Settings → App Permissions → Allow notifications
3. Quiet hours not active
4. Notification preferences for specific type enabled

### Bus Location Not Updating?

**Possible reasons:**
- Bus helper hasn't started tracking
- GPS signal lost
- Network issue

**Solution:**
- Wait a few minutes
- Refresh the screen
- Contact school if issue persists

### Cannot Make Payment?

**Check:**
- Payment gateway is working
- Card details are correct
- Sufficient balance
- Bank allows online transactions

**Alternative:**
- Try different payment method
- Pay via school office
- Contact school admin

## Security & Privacy

### Keep Your Account Secure

1. **Strong Password**
   - At least 8 characters
   - Mix of letters, numbers, symbols
   - Don't share with anyone

2. **Enable Biometric Login**
   - Settings → Security
   - Enable Face ID / Fingerprint
   - Faster and more secure

3. **Logout from Shared Devices**
   - Always logout when using shared devices
   - Don't save password on public devices

### Privacy

- Your child's data is encrypted
- Only you can see your child's information
- Teachers can't see other students' parent view
- School admin has limited access

## FAQs

### Q: Can I add another parent to the account?
**A:** Contact school admin to link multiple parents to a child.

### Q: How do I know if my child boarded the bus?
**A:** You'll receive a notification when the bus helper marks your child as boarded.

### Q: Can I see other children's marks?
**A:** No, you can only see your own child's data for privacy.

### Q: How often is data updated?
**A:** Real-time for transport, attendance marked within 30 minutes of class start, homework and marks updated immediately.

### Q: What if I forget my password?
**A:** Tap "Forgot Password" on login screen, enter email, check email for reset link.

### Q: Can I use the app offline?
**A:** Limited features work offline. View cached data, but can't fetch new data or submit forms.
